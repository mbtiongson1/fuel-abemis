     set if it fails (already installed).

      the slowest. If it tries to re-extract all PDFs, restrict it to a no-op
     pass by reading the existing AMTEC_full_extracted_dataset.xlsx. Inspect

      from the failing cell using --from <idx>-style invocation or by running
     individual cells with nbconvert.
